# Lifecycle Analysis Platform

Enterprise hardware inventory lifecycle analysis system with AI-enhanced EOL prediction.

## Features
- CSV inventory upload and parsing
- AI-powered lifecycle date research
- Risk assessment and reporting
- Excel export with year-by-year analysis

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL
- AI: Google Custom Search API

## Deployment
Configured for Google Cloud Run with Cloud SQL PostgreSQL.
